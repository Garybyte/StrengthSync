# Strength Sync

Strength Sync is a MERN stack fitness tracking online application that lets users set up accounts, record workouts, measure fitness improvement over time, and track nutrition objectives.

## Features

- User authentication with JWT
- Protected routes
- Workout tracking
- Nutrition tracking
- Personalized calorie recommendations
- Progress tracking
- MongoDB database integration
- Full CRUD functionality
- User-specific data storage

## Tech Stack

Frontend:
- React
- React Router
- Axios
- CSS

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
